### Try

- Install nodejs -> https://nodejs.org/en/
- Install rust         -> https://www.rust-lang.org/tools/install
- Install wasm-pack -> https://rustwasm.github.io/wasm-pack/installer/

```
git clone https://github.com/UlysseCorbeil/rust-wasm
cd rust-wasm/
wasm-pack build
cd www
npm i
npm run start
```
